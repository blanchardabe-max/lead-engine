Connect Lusha
This turns "Add Lead" and "Enrich" into automatic lookups instead of manual typing — pull a name + company, get back a verified email/phone.
Why "proxy" mode exists: Lusha's own docs say API keys must stay server-side, and most browsers block direct calls to api.lusha.com from a local file anyway (CORS). Direct mode is here so you can quickly see if your key works at all — but for daily use, deploy the free relay script below and switch to Proxy mode. Your key then lives only in that relay, never in this page.
Deploy the relay (one-time, ~5 min, free)
Cloudflare Workers free tier. No server to maintain.
  1. Go to workers.cloudflare.com → sign up free → "Create Worker"
  2. Delete the default code, paste in lusha-proxy-worker.js (downloaded alongside this tool)
  3. In Worker Settings → Variables, add a secret: LUSHA_API_KEY = your real Lusha key
  4. Click Deploy → copy the worker URL it gives you (looks like https://lusha-proxy.yourname.workers.dev)
  5. Paste that URL into Proxy URL above, save, hit Test
